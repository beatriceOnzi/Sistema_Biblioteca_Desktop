## Funcionalidades

- Cadastro de alunos
- Cadastro de livros
- Controle de empréstimos e devoluções
- Organização por turmas
- Consulta de registros da biblioteca
- Consulta de histórico de empréstimos

## Tecnologias

- Python
- Tkinter
- SQLite

## Como utilizar

1. Clone ou baixe o projeto.

2. Execute o arquivo principal:

```bash
python main.py
```

Ou abra o arquivo:
Biblioteca.bat
